Top-most position

Above Admission Summary
2. Purpose of “School List”
The School List acts as a centralized control panel for the Super Admin to manage all schools associated
with the ERP system.
It includes:
Schools where the ERP is genuinely sold and deployed
Dummy / Demo schools created for testing, demos, and sales presentations
3. School List Contents
The School List should display the following fields:
School Name
School Type : Small / medium / Large
Package : Money xxxx(RS)
Created Date

7. Key Summary
School List is the master control for all ERP-linked schools
Clicking any school opens its GiNi School Dashboard (HOME)
Super Admin has full dashboard data control for:
Genuine schools
Dummy schools
Deletion is allowed only for Dummy Schools
Genuine schools are protected from deletion
